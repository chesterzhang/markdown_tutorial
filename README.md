## 8.列表语法
```python

#无序列表语法
- 目录1
- 目录2
- 目录3
#注意 - 后面要跟一个空格，若不跟一个空格表示下一级目录

#有序列表语法
1. 名称 
#数字1可以替换成别的数字
#数字后面跟一个空格表示和上级同一个目录，若不跟一个空格表示下一级目录

```

无序列表
- 目录1
- 目录2
- 目录3

有序列表
1. 目录1
2. 目录2
